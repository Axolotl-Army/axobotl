### MEDIUM

**M1 -- `document.write()` in app-footer.ejs (line 4)**
- `document.write(new Date().getFullYear())` is an anti-pattern
- Fails silently with async/deferred loading; triggers browser deprecation warnings
- Fix: Replace with `<%= new Date().getFullYear() %>`
- Effort: 1 minute

**M2 -- Unbounded pagination in logs.ejs (lines 74-78)**
- Renders every page number as a link: `for (let i = 1; i <= pagination.total; i++)`
- With 500+ pages, generates hundreds of DOM elements
- Fix: Implement smart pagination (show first, last, current +/- 2, with ellipsis)
- Effort: 10 minutes

### LOW

**L1 -- 42 unused page scripts in `public/scripts/pages/`**
- Only `auth-animation.js` is loaded by any EJS template
- 42 other files are SmartAdmin demo page scripts (chart demos, calendar, etc.)
- Total bloat: ~680KB
- Fix: Delete all except `auth-animation.js`
- Effort: 1 minute

**L2 -- Discord avatar URL logic duplicated in app-header.ejs**
- Avatar URL construction repeated at lines 57-60 and 68-71
- Minor DRY violation; could extract to server-side middleware
- Fix: Add `avatarUrl` to `res.locals` in auth middleware
- Effort: 5 minutes
